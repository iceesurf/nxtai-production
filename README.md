# NXT.AI - Automação Inteligente

## 🚀 Sobre
Plataforma SaaS completa de automação empresarial com WhatsApp Business API e IA.

## 💻 Tecnologias
- Firebase (Hosting, Functions, Firestore)
- WhatsApp Business API
- OpenAI GPT-4
- Node.js 18+

## 🛠️ Instalação
```bash
npm install
npm run dev
```

## 📦 Deploy
```bash
npm run deploy
```
